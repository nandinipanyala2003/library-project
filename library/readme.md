# 📚 Library Management System (Python)

A simple and interactive **Command-Line Library Management System** built using Python.  
This project allows users to register, login, and manage books efficiently through a menu-driven interface.

---

## 🚀 Features

- 👤 User Registration & Login System  
- 📚 View All Books  
- 🔍 Search Books (by Title, Author, ID)  
- ➕ Add New Books  
- 📖 Issue Books  
- 👤 View User Profile  
- 💾 Data stored using JSON  

---

## 🛠️ Tech Stack

- Python 🐍  
- JSON (for data storage)  
- OOP Concepts
